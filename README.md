# Simple-Tag HTML
The most commonly used HTML tags are as follow :
```
< p > : The tag p defines a paragraph 
< strong > : Defines important text 
< em > : Emphasized text 
< small > : Defines small text 
< h1 > : For sitetitle 
< h2 > : For webpage 
< h3 > : For anarticle heading 
< h4 > : For a subnavigation inside thearticle 
```
